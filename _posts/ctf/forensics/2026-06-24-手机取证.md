## Android

#### 数据目录

应用目录：/data/app，base.apk（安装包）

应用数据目录：/data/data/xxx.xxx.xx

- 数据库：Databases，微信：数据库：MicroMsg/EnMicroMsg.db
- 文件资源：Files
- 配置文件：share_prefs

/data/system，记录手机软件、Widget（小部件）等信息，锁屏密码的密钥

/data/misc，保存WIFI账号和VPN等配置信息的目录

#### 常用ADB命令

```sh
adb shell dumpsys activity top | findstr ACTIVITY # 当前在运行APP（包含后台）
adb shell dumpsys window | findstr mCurrentFocus # 当前focus的APP
```

navicat，工具 - 》在数据库或模式中查找

外部存储：/storage/emulated/0

#### 提取镜像

###### 屏幕锁

开发者模式：关于，版本号（长按），开发者选项，
打开：USB调试，不锁定屏幕
关闭：监控ADB应用安装，允许模拟点击，不保留活动，启用MIUI优化

###### 连接模式：文件传输模式、MIP

1、root权限，已知（绕过）密码：逻辑提取
2、非root，已知（绕过）密码：备份提取
3、漏洞：芯片镜像

###### 备份提取：

- 系统自备份：原厂自带，近似root逻辑提取，无风险
- 互传、换机、克隆

ROM芯片分成8个区：

1. hboot分区：负责启动
2. radio分区：负责驱动
3. recovery分区：负责恢复
4. boot分区：系统内核
5. system分区：系统文件
6. cache分区：系统缓存
7. userdata分区：用户数据
8. SD卡：用户额外存储空间

#### [Huawei备份文件解密](https://github.com/RealityNet/kobackupdec)

将备份文件夹下所有文件还原出来，再解密。（sample：内存分析-longjiancup-2021）

```
python3 D:\ctf\tools\forensics\kobackupdec\kobackupdec.py -vvv password inputdir outputdir
```

## IOS

programData/Lockdown 连接过的iphone信息

IPA，打开info.plist文件，搜索bundleidentifier获取包名

备份文件加密，Manifest.plist文件导入passware，解密

短信：/private/var/mobile/Library/SMS/sms.db

旧版：`/private/Application.plist`，`/private/var/mobile/Container`，配置目录：Library，右键 - 》跳转到源文件

新版（Iphone15）：包名

系统分区： /

- /Applications，系统预装的应用
- /etc，链接至 / private / etc
- /Developer，包含系统插件和设置
- /System，包含系统组件、内核模块等

用户分区： / private/var

- /logs，系统日志
- /mobile，用户数据
- /Library，通讯录等
- /Media，照片等
- /Containers/Data / Application，第三方软件
- /MyApp.app，应用本身数据
- /Documents，关键数据
- /Library，配置文件等：Preferences应用默认设置；Caches可再生数据，如网络请求
- /tmp，临时文件
- /wireless，CellularUsage.db记录用户使用过的sim卡信息
- /keychains，Keychains-2.db，包含各种应用程序的用户密码
- /data/misc/adb/adb_keys曾授权连接该手机的电脑的ADB公钥
